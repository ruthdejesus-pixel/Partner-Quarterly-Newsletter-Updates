FME 2026.2 is now available Now Live
AI integration, workflow design, and platform administration improvements
View details ▼

FME 2026.2 is now available! This release introduces new and improved enhancements across AI integration, workflow design, and platform administration.

  • FME Flow as an MCP Server — Expose FME workspaces as tools that AI agents can discover and invoke through the Model Context Protocol (MCP), connecting enterprise data and workflows directly to AI-powered experiences.
  • Native looping in workspaces — A long-requested capability now built directly into workspace design.
  • New data capture capabilities in FME Realize — Expanded options for capturing and working with data in the field.
  • Expanded JSON support + platform improvements — Plus a range of integration and usability improvements across the platform.
⚠️ Action recommended — Security Tiers FME 2026.2 introduces new security tiers that may affect some existing workflows after upgrading. Most customers won't notice a difference, but certain FME components may behave differently depending on the security tier enabled. We recommend reviewing this article before upgrading. If customers run into issues, direct them to Safe Software Support.

Licensing
Trial licenses — new process Updated
Trial licenses are now issued via serial number for improved tracking

We've updated how trial licenses are issued. Partners must now create a Deal in HubSpot rather than sharing a license file directly — this ensures consistent tracking and streamlined fulfillment across all trial requests.

  • Deal name must include "Trial" — The word "Trial" must appear in the Deal name, and Trial Deals must be used only for trial licenses. Do not combine with any other license type.
  • Select "Trial" as the Use Type in DealHub — Then select the Trial Duration to define how long the trial asset will remain active — 30 or 60 days.
  • All standard Deal fields still required — Billing Contact, Shipping Contact, and Primary AMC Contact are all mandatory.
  • Automated serial number notification — Once the order is processed, the Shipping Contact on the Deal will automatically receive the serial number(s) — the same process used for all other license orders.
